# 「開発を応援」ダイアログ aligning 計画書

## 1. 目的
「開発を応援」ダイアログの内容とデザインを、v1（旧エディション）と完全に一致させ、v2のコンポーネントとして最適化する。

## 2. 現状の分析
- **v1の内容**: `index.html`内の`donate-dialog`に詳細な説明文、Special Thanks（寄付者リスト）、寄付ボタン（Stripeへのリンク）、開発者プロフィール、開発の経緯が含まれている。
- **v2の現状**: `DonateDialog.tsx`は非常にシンプルで、テキストやセクションが不足しており、寄付先リンクも異なっている。

## 3. 要件
- **文言の完全一致**: 見出し、説明文、Special Thanks、寄付ボタン、プロフィール、開発の経緯のすべてを一字一句違わずv1から移行する。
- **デザインの再現**: v1のセクション構成やアイコン使いをTailwind CSSで再現する。
- **実装ルールコード**: `rules.md`に基づき、以下の方針で実装する。
  - スタイリングは Tailwind CSS のみを使用する。
  - テキストはコンポーネント内に直接記述する（ユーザー指示を優先）。
  - アイコンは `lucide-react` を使用する。
  - UI部品（BaseDialog等）を適切に使用する。
