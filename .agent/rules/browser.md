1. 既存タブの再利用 (Minimize Tab Proliferation) ブラウザ操作を行う際は、毎回新しいタブを開くのではなく、対象のURL（特にローカル開発サーバーなど）が既に開かれていないか確認し、可能な限り既存のアクティブなタブを再利用して操作を行ってください。

2. 検証前のハードリロード (Hard Reload Before Verification) UIの変更を確認する際やスクリーンショットを撮影する直前には、必ず Ctrl+Shift+Rを送信してハードリロードを実行してください。キャッシュによる古い状態での誤った検証を防ぐため、リロードの完了を待ってから次のアクションに移ることを徹底してください。
